# Carbon Emission Dashboard-Excel

An interactive Excel dashboard built to analyze global CO₂ emissions across countries and regions from 1990 to 2019.

# Dashboard Features

- Top 10 highest CO₂-emitting countries
- Regional comparison of total emissions vs average emissions per capita
- Year-wise carbon emission trend analysis
- Regional distribution of average metric tons per capita
- Interactive filters for Year and Region
  
# Tools used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design & Data Visualization
  
- # Key Findings
  
- China is the largest CO₂ emitter in the dataset, followed by the United States.
- Russia, India, and Japan** are among the other major contributors to global emissions.
- Total global CO₂ emissions show a steady upward trend from 1990 to 2019.
- Asia records the highest total CO₂ emissions among all regions.
- Europe has the highest average CO₂ emissions per capita.
- Africa contributes the lowest average emissions per capita.
  
- # Project Objective
  
- The goal of this project is to explore carbon emission patterns across countries and regions, identify major contributors, and present insights through an easy-to-use interactive Excel dashboard.
